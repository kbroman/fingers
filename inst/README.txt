This is a quick document to explain the installation and use
of the "fingers" package for R


Installation (Windows)

  1. Unzip the "finger.zip" file into the directory $RHOME\library
     ($RHOME is something like c:\Program Files\R\rw1000)
     Note that this should create a directory $RHOME\library\fingers 
     containing the R source code and the compiled dll

  2. Start Rgui 

  3. Type "link.html.help()" to get the help files for the fingers
     package added to the help indices

  4. Note that the source code is in the file "fingers_*.tar.gz"


Installation (Unix)

  1. Go into the directory containing "fingers_*.tar.gz"

  2. Type "R INSTALL qtl"
